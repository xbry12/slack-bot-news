## Prerequisites 
For this project you will need 

* node version v19.3.0

### Installation
1. Clone this repo
``` sh
git clone https://github.com/xbry12/slack-bot-news.git
```
3. Install NPM Packages 
``` sh 
npm install 
```
``` sh
npm init 
 ```

4. Enter your API in config.js
``` sh
const API_KEY = 'Enter your API KEY';
```

5. Start the app 
node app.js
